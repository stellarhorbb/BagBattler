STRIKE
ATTACK
Deals base ATK damage to the Entity.
Stacks with every other Strike on the line.
GUARD
DEFENSE
Absorbs base DEF from incoming
Entity damage.
When 2 GUARD are aligned, value ×1.5
PROVOCATION
MODIFIER
Reduces Entity DMG by 25%.
Reduces Entity DMG by 75%
if placed on the first slot of the line.
RAMPART
DEFENSE
Doubles every DEF token on the line
when placed as the last token.
Must be the last slot on the line
with at least one DEF token before it.
HAZARD
HAZARD
Drawn automatically and placed on a
random free slot. Two Hazards on the line
triggers a CRASH.
